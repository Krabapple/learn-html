TIL that:

  1. <input> can be used to describe a text box, a submit button, a check box, a radio button etc.
  2. Use value inside the input tag to change the name of the button.
  3. Use <label> to tell the user what to enter.
  4. To associate the label with the input box, use for="" and ID.
    (And then if you click the label, the input box will become active.)
  5. Use unordered list tag to wrap the boxes to let them look nicer.
    Don't use <p> since <input> and <label> are not its childern.
  6. If you want to set the width and length for the comment box, you can use col and row, but it is not a good practice.
    Style them in CSS instead.
  7. Make the form method "get" and the information will go through the url.
    Then if you click submit, notice the url:
    ? for name and & separate the names.
  8. If the checkbox is checked, then we have a "on" in the url.
    If the checkbox is not checked, nothing will be sent through the url. (No off only on)
  9. You can not uncheck the radio button.
