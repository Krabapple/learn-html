    Two ways to create a sidebar:

  • Use <div class="sidebar">. (Usually seen in old webpages)
    Not a good practice because div does not have semantic meanings. They are simple wrappers.
  • Use <aside>.

TIL that:

  1. You can style multiple classes by writing them together.
    ".content, aside{ ... }"
  2. Only want to style the ul in the sidebar?
    Use "aside ul { ... }"
  3. Using precentage to define the width of the boxes, then you can change their look by adjusting your browser.
    And don't forget to set a min-width, otherwise when the browser's width is too small, the text will go out of the box.
  4. When we apply float, it take these elements out of the flow of the documents.
  5. When applying background color for the wrapper, it seems that the color only reach the bottom of the title. The solutions are:
    • Add a footbar
    • Clear all the floats by using clear
    • Within wrap - overflow: hidden
