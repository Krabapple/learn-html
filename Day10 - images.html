TIL that

  1. Use src instead of href in <img>.
  2. Always write a alt for an image in case the image is crash.

  3. Do not reference images from a actual url link! It will cause the url owner to pay for displaying the image on your webpage, which is not good. (This is called hot linking)

  4. Save them to lacal and use them instead.
  5. In summary: invert an image = <img> + scr + alt
