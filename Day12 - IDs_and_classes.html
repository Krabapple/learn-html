TIL that

  1. IDs and classes are simple ways to assign labels to specific elements and then target them within style sheets.
  2. You can put a <style> in <head> if the style thing is small and don't need to create a style sheet.
  3. Three ways to do a small style design:
    • Use an identifier: id=" ". Don't use a name like "third_line" in case it moves to another line later.
      (Note that we use IDs to deal with specific elements. Do not apply them to a group of elements.)
    • Use classes to style groups of elements.
      • (In <head>, use # for IDs and . for classes.)
    • Use li:nth-child.
