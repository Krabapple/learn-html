    We can re-design the color by changing the style:

  • style="color: green;" <-- simple color
  • style="color: rgb(100, 200, 50);" <-- rgb color
  • style="color: #ff00cb;" <-- hex color


    We can overwriting the default style, and what the browser shows will be our own style:

  • For example, linking to Google without underline and with orange color.
  • If you view the source code of the line above, you will see <style="color: orange; text-decoration: none;">, which is really messy.
  • So try to create a style sheet! Create a style sheet with the extension name ".css", and link to it in <head>.
