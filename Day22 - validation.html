TIL that:

  1. Browsers nowadays are smart enough to fix the bad coding for you. But you still need to check.
  2. So use validator! A validator will make sure if we've done everything correctly.
  3. validator.w3.org
  4. Don't blindly follow the validator - it's not a rule. If you know what you're doing, then keep going.
  5. Check between the validator and your html file. If the warnings/errors the validator shows are fine in html, just skip it.
  6. While doing a project, check in the validator few hours or so to make sure everything is fine.
  7. When your CSS doe not take effect, first CHECK IN THE VALIDATOR.
