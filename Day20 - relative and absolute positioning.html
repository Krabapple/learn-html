TIL that:

  1. Write "position: relative;" in your CSS and set the top, left, right, bottom margin value for the box.
    In "position: relative;" mood, top = push down, bottom = push up, right = push left, left = push right.
    We have the red box as an example of "position: relative;".
  2. However "position: absolute;" does a quite different thing.
    It starts from the upper-left corner of the browser, and instead of "pushing", it "locates" the box.
    We have the blue box as an example of "position: absolute;".
  3. All the locations are related to the nearest parent.
  4. See the blue box inside the wrap:
    100px is no longer 100px from the left of the browser window, but 100px from the wrap.
    But if we remove the "position: absolute;" from the CSS of wrap, the nearest parent of the blue box will become body, or the browser window, and the box will be located in the bigger wrap, that is, the body.
  5. Use absolute positioning when you want some little thing exceeds the boundary of the container.
    (like the pink box and red heart on the left side of the wrap)

  6. In summary:
    A relative positioning gives us the ability to specify top, left, right and bottom values. A absolute positioning locates the point not relatively, but straight forward. (How to make it clear? Just try to understand it...)
